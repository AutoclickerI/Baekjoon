<h2>채널 구독</h2>
</div>

<p>마음에 드는 채널을 구독하면 해당 채널의 더 많은 콘텐츠를 볼 수 있습니다. YouTube 동영상 아래 또는 채널 페이지에서 구독 버튼을 찾을 수 있습니다. 채널을 구독하면 새로 게시되는 동영상이 구독 피드에 표시됩니다.</p>

<p>구독한 채널에서 새 콘텐츠를 게시할 때 알림을 받을 수도 있습니다. 기본적으로 채널의 하이라이트에 대한 알림만 전송됩니다. <a href="https://support.google.com/youtube/answer/3382248">알림 관리 방법</a>을 자세히 알아보세요.</p>

<p>채널을 구독 취소한 다음 다시 구독하면 알림 설정이 재설정됩니다.</p>

<p>채널 시청자층이 '아동용'으로 설정된 경우에는 알림이 전송되지 않습니다. 또한 종 모양 알림 아이콘이 알림 없음 <img alt="" height="20 " src="https://lh3.googleusercontent.com/53nydMFlyzLUiKizGsbIgh3ZP8qp6ZFGCx_lwKcedCLExGX75KvfuU97ydL2ALuc8w=w40-h40" width="20">으로 설정됩니다. 이 설정은 변경할 수 없습니다. </p>
